# Console Educational System

This project implements a simple console-based educational system demonstrating OOP principles, modularity, decorators for access control and logging, and file-based persistence (JSON/CSV/TXT).

Run:

```bash
python3 main.py
```

# Talabalar Boshqaruvi Tizimi (Konsol)

Konsol asosidagi Talabalar Davomati va Baholarini Boshqarish tizimi. Loyihada OOP, dekoratorlar va faylga asoslangan saqlash misol qilib keltirilgan.

Ishga tushirish:

```bash
python3 main.py
```

Bog'lanmalar o'rnatish:

```bash
pip install -r requirements.txt
```

Fayllar:
- `main.py`, `models.py`, `storage.py`, `decorators.py`, `ui.py`, `reports.py`
